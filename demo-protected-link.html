How It Works

  1. Upload: You upload your Lua code as a .txt file
  2. Protection: We store it securely behind access controls
  3. Distribution: You get a protected loadstring link
  4. Execution: Authorized users can execute, unauthorized see "ACCESS DENIED"
  5. Control: You manage who can access your scripts
